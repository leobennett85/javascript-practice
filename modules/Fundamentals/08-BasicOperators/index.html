Basic Operators, Maths

Terminology: "unary," "binary," and "operand"

Operand

What 'operators' are applied to in expressions.

The expression 5 * 2 has 2 operands: "5" and "2".

Unary

Operator is unary if it has a single operator.

x = -x; is unary

Binary

Operator is binary if it has 2 operands

Maths

Self explanitory: + - * /

Remainder: %

Exponentiation: **

Remainder

The remainder operator returns the remainder of deviding the operands

5 % 2 = 1, the remainder of 5/2 = 2R1

Exponentiation

** raises the first operand to the power of the second operand

2 ** 2 = 4 (2^2)

50 ** 8 = 39062500000000
