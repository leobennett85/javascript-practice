10-Conditionals

The "if" statement

Use the if statement to check conditions.

let year = prompt('What year was Leo born?", '');

if (year == 1985) alert('You are right!');

Else clause

Use the else clause to execute when the condition is false

if (year == 1985) {
alert("You got it!");
} else {
alert("WRONG!"); }

Elseif

Use Elseif when you have several specific conditions to check

if (year < 1985) {
alert("Too early b'y!");
} else if (year > 1985) {
alert("Too late b'y!");
} else {
alert("Exactly!");
}

Conditional operator '?'

Sometimes we assign a variable based on conditions

let accessAllowed;

let age=prompt('How old is ya?', '');
if (age>18) {
accessAllowed = true;
} else {
accessAllowed = false; }
alert(accessAllowed);

The question mark operator (ternary) allows us to do this in an easier way.

The syntax is as follows:
let result = condition ? value1 : value2;

The previous code can be re-written as:

let accessAllowed = age > 18 ? true : false;
